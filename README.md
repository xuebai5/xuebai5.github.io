# SkyServer
### C++17 cross-platform realtime server
- Windows ubuntu测试通过
- CMake组织工程
- 工作在传输层/网络层(TCP/IP)
- Asio实现网络跨平台(IOCP/EPOLL)
- 同时支持TCP/WEBSOCKET连接
- Accept/NetIO线程可配置
- Protobuf格式网络协议
- MySQL存储冷数据
- Redis存储热数据
- Recast自动寻路
- Yaml格式配置
- 使用Curl进行HTTP访问
- 订阅发布模式消息分发器
